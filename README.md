Manufacturing Operations SQL Analytics

Project Overview

This project focuses on solving real-world manufacturing and operational business problems using SQL.

The analysis covers:

- Machine downtime
- Production efficiency
- Quality defects
- Supplier delays
- Inventory risk
- Plant performance evaluation

The project was designed to simulate practical business analytics scenarios commonly faced in manufacturing industries.

---

Objectives

- Identify operational bottlenecks
- Analyze production losses
- Evaluate supplier reliability
- Monitor inventory risks
- Improve plant-level efficiency tracking
- Generate business insights using SQL analytics

---

Tools & Concepts Used

- SQL
- Common Table Expressions (CTEs)
- Window Functions
- Aggregations
- CASE Statements
- Ranking Functions
- KPI Calculations
- Operational Analytics

---

Business Problems Solved

1. Machine Operational Loss Analysis

Identified machines contributing to the highest operational losses based on:

- Downtime events
- Total downtime
- Production loss
- Production efficiency

KPIs Used

- Total Downtime
- Average Downtime
- Production Efficiency %
- Difference in Production Output

---

2. Plant & Shift Quality Defect Analysis

Analyzed which plants and shifts generated the highest number of defects during production.

KPIs Used

- Total QC Checks
- Total Defects
- Shift-wise Defect Ranking

---

3. Supplier Delay & Reliability Analysis

Evaluated suppliers causing the highest operational delays and calculated actual reliability scores.

KPIs Used

- Delay Frequency %
- Average Delay Days
- Supplier Reliability Score
- Delayed Orders Count

---

4. Inventory Reorder Risk Classification

Categorized inventory risk levels across warehouses based on stock coverage and reorder levels.

Risk Categories

- High Risk
- Moderate Risk
- Low Risk

KPIs Used

- Stock Coverage %
- Risky Products Count
- Warehouse Risk Ranking

---

5. Plant Performance Score Analysis

Created a custom plant performance scoring model combining:

- Production efficiency
- Downtime impact
- Quality defect impact

KPIs Used

- Production Efficiency %
- Total Downtime
- Total Defects
- Plant Performance Score

---

Key Insights

- High downtime machines directly impacted production efficiency.
- Certain shifts consistently showed higher defect rates.
- Supplier delays significantly affected operational continuity.
- Some warehouses operated close to critical reorder levels.
- Plants with balanced efficiency and lower defects achieved higher performance scores.
